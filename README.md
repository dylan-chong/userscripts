# Userscripts

A collection of [Tampermonkey](https://www.tampermonkey.net/) userscripts.

## Installation

1. Install the [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. Click an install link below — Tampermonkey will prompt you to install the script
3. Scripts auto-update when a new version is pushed to this repo

### Mac / iOS

Use the [Userscripts](https://apps.apple.com/us/app/userscripts/id1463298887) app to run scripts on iOS and Mac with iCloud sync:

1. Install the Userscripts app on both platforms
2. Clone this repo inside an iCloud Drive directory
3. Set the Userscripts directory on both platforms to the iCloud folder
